# Let's Study MS - School of Mathematics

This is the official web application for **Let's Study MS**, a leading mathematical institute specializing in higher education coaching (IIT JAM, CSIR-NET, GATE, ISI, TIFR, etc.).

## 🚀 Tech Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **SEO**: React Helmet

## 🛠️ Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (LTS version recommended).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/gargs64/Lets-Study-MS.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Lets-Study-MS
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the local development server:
```bash
npm run dev
```
The application will be available at `http://localhost:3000`.

### Production Build

To create a production-ready bundle:
```bash
npm run build
```
The output will be generated in the `dist/` directory.

### Preview Production Build

To test the production build locally:
```bash
npm run preview
```

## 📁 Project Structure

- `src/components/`: Reusable UI components (Header, Footer, etc.)
- `src/pages/`: Main page components (Home, Testimonials, Courses, etc.)
- `src/pages/courses/`: Detailed course pages
- `src/assets/`: Images and other static assets
- `public/`: Public assets like `robots.txt` and `sitemap.xml`

## 🌐 Deployment

This project is optimized for deployment on platforms like **Vercel** or **Netlify**.

### Deploy to Vercel (Recommended)
1. Connect your GitHub repository to Vercel.
2. Vercel will automatically detect the Vite config.
3. Set the build command to `npm run build` and publish directory to `dist`.
